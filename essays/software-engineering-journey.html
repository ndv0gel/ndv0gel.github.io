Software Engineering Journey

Software engineering is such a dynamic intersection of problem-solving, creativity, and technological innovation that is always so fascinating for me. I am most interested in C programming and Typescript, but especially in how modern applications can scale seamlessly across multiple regions while maintaining strong data consistency and reliability.

Above and beyond the technical infrastructure, I have a big interest in the development of my skills concerning data engineering. With AI continuous integration in software development, the ability to effectively develop, deploy ML models, feature stores management, and create scalable data pipelines definitely proves to be an added advantage.
